Ausrüstung

  • 3.1.Es darf jede Flugscheibe verwendet werden, auf die sich die beiden Mannschaftskapitäne einigen.
  • 3.2.Jeder Spieler muss Kleidung tragen, an der man seine Mannschaftszugehörigkeit erkennen kann.
  • 3.3.Kein Spieler darf Kleidung oder andere Ausrüstungsgegenstände tragen, die eine erhöhte Verletzungsgefahr begründen.
